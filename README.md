# react-native-printer-brothers

Package for using RJ4230B printer of brothers

## Installation

```sh
npm install react-native-printer-brothers
```

## Usage

```js
import { multiply } from "react-native-printer-brothers";

// ...

const result = await multiply(3, 7);
```
